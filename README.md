Rock–Paper–Scissors 
Project Description
A simple command-line implementation of the classic Rock–Paper–Scissors game in Python. The user plays against the computer by selecting a move; the computer picks randomly, and the winner is determined according to traditional rules.

Game Rules
Rock vs Paper: Paper wins

Rock vs Scissors: Rock wins

Paper vs Scissors: Scissors wins

How to Play
Make sure you have Python 3 installed on your system.

Download or copy the code to a file named rock_paper_scissors.py.

Run the script in your terminal:

text
python rock_paper_scissors.py
Enter your move by typing:

1 for Rock,

2 for Paper,

3 for Scissors.

The program will tell you the computer's move and the winner.

After each round, you can choose to play again.
